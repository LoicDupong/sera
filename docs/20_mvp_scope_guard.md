# 20 — MVP Scope Guard

## What must stay small
- Profil host : email + mot de passe uniquement. Pas de photo, bio, préférences, ou settings avancés.

## What to reject in MVP
- Page de settings host complexe
- Gestion de multiple profiles ou organisations
- Historique d'activité détaillé
- Notifications in-app / feed
- Tout ce qui ne sert pas directement le flow créer un event → inviter → tracker les réponses

## Decision rule
If a feature does not directly improve the core user flow of the MVP, it should be postponed.
