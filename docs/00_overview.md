# 00 — Overview

## Project Name
Sera

## One-line description
App de gestion d'événements permettant à un host de créer un event, gérer sa liste d'invités et envoyer un lien pour que les invités puissent répondre facilement

## Value proposition
Sera simplifie radicalement l'organisation d'événements privés en éliminant le chaos des groupes WhatsApp et des réponses dispersées.

## Why this app should exist
Une seule app pour inviter, suivre les réponses en temps réel et garder tout clair — sans friction pour les invités, sans compte requis de leur côté.
