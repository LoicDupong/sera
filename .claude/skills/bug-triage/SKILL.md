# Bug Triage

## Process
1. Identify the exact bug
2. Confirm probable root cause
3. Estimate impact
4. Propose the smallest safe fix
5. Flag regressions to test

## Output format
- Symptom
- Root cause hypothesis
- Fix recommendation
- Validation checklist
