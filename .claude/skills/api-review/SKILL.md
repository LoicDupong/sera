# API Review

## Check
- route naming consistency
- public vs protected separation
- payload clarity
- validation needs
- unnecessary endpoints
- alignment with docs/02_mvp_features.md and docs/10_models_spec.md

## Output format
- Good
- Risks
- Missing pieces
- Suggested simplifications
